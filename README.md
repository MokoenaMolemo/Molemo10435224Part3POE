# Claims Management System

A web application for managing employee expense claims with multi-level approval workflow.

##  Features

### User Roles & Workflow
- **Employee**: Submit expense claims with supporting documents
- **Coordinator**: Initial review and validation of claims
- **Manager**: Final approval/rejection of claims
- **HR**: Settlement processing and reporting

### Core Functionality
-  Multi-level claim submission and approval workflow
-  Document upload and management (receipts, supporting files)
-  Real-time dashboard for each role
-  Advanced reporting and analytics
-  Settlement processing with payment tracking
-  Secure authentication and authorization

##  Technology Stack
### Backend
- **ASP.NET Core MVC** - Web framework
- **Entity Framework Core** - ORM and data access
- **SQL Server** - Database
- **Dependency Injection** - Service architecture

### Frontend
- **Bootstrap 5** - UI framework
- **Razor Pages** - Server-side rendering
- **jQuery** - Client-side interactions
- **Chart.js** - Data visualization

### Security
- **Cookie-based Authentication**
- **Role-based Authorization**
- **Input Validation**
- **Anti-forgery Tokens**
